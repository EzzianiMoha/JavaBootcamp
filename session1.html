Setting Up Your Development Environment

Installing JDK (Java Development Kit)

The JDK is essential for Java development as it includes the Java Runtime Environment (JRE) and development tools like the Java compiler.

Step 1: Visit the Oracle JDK download page or OpenJDK website to download the latest version of JDK.

Step 2: Follow the installation instructions for your operating system (Windows, macOS, or Linux).

Step 3: Verify the installation by opening a terminal/command prompt and typing:

java -version

You should see information about the installed Java version.

Installing IntelliJ IDEA

IntelliJ IDEA is a powerful Integrated Development Environment (IDE) for Java development.

Step 1: Visit the IntelliJ IDEA download page.

Step 2: Download the Community Edition (free) or Ultimate Edition (paid with free trial).

Step 3: Follow the installation instructions for your operating system.

Step 4: Launch IntelliJ and create your first Java project:

  1. Click "New Project"
  2. Select "Java"
  3. Choose your JDK version
  4. Click "Create"

Java Basics

Every Java application must have a main method, which is the entry point of the program.

Variables in Java

Variables are containers for storing data values. In Java, you must declare a variable before using it.

Basic Data Types

Integer (int)

Used for storing whole numbers without decimals.

int age; // Declaration age = 25; // Assignment int year = 2023; // Declaration and assignment int quantity = 10;

If you try to use a variable without initializing it, you'll get a compilation error:

int age; System.out.println(age); // Error: variable age might not have been initialized

Double

Used for storing numbers with decimal points.

double price = 19.99; double temperature = 36.6; System.out.println("The price is $" + price); // Concatenation

Character (char)

Used for storing single characters.

char grade = 'A'; char symbol = '$'; char currency = '‚Ç¨';

Boolean

Used for storing true/false values.

boolean isStudent = true; // Note: camelCase naming convention boolean forSale = false; boolean isOnline = true;

String (Reference Type)

Used for storing sequences of characters (text).

String name = "John"; String food = "Pizza"; String email = "john@example.com"; System.out.println("Hello " + name); System.out.println("Your email is " + email);

Accepting User Input with Scanner

To get input from the user, we use the Scanner class from the java.util package.

import java.util.Scanner; public class Main { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Enter your name: "); String name = scanner.nextLine(); // For strings with spaces System.out.println("Hello " + name); scanner.close(); } }

Reading Different Data Types

System.out.print("Enter your age: "); int age = scanner.nextInt(); System.out.print("How much is your jacket? "); double jacketPrice = scanner.nextDouble(); System.out.print("Are you a student? (true/false): "); boolean isStudent = scanner.nextBoolean(); System.out.println("You are " + age + " years old"); System.out.println("Your jacket costs $" + jacketPrice); System.out.println("Student status: " + isStudent);

‚ö†Ô∏è Common Issue: nextLine() after nextInt()

When using nextInt() followed by nextLine(), you might encounter an issue where nextLine() doesn't wait for input.

System.out.print("Enter your age: "); int age = scanner.nextInt(); System.out.print("Enter your favorite color: "); String color = scanner.nextLine(); // This might not work as expected System.out.println("You are " + age + " years old"); System.out.println("You like the color " + color);

Explanation: The nextInt() method reads only the integer, leaving the newline character in the buffer. The nextLine() then reads that newline as an empty string.

Solution: Add an extra scanner.nextLine() after nextInt() to consume the newline character.

System.out.print("Enter your age: "); int age = scanner.nextInt(); scanner.nextLine(); // Consume the newline character System.out.print("Enter your favorite color: "); String color = scanner.nextLine(); // Now this will work correctly

Arithmetic Operators

Arithmetic operators allow you to perform mathematical calculations in your programs.

Basic Arithmetic Operators

int x = 10; int y = 2; int z; z = x + y; // Addition: 12 z = x - y; // Subtraction: 8 z = x * y; // Multiplication: 20 z = x / y; // Division: 5 z = x % y; // Modulo (remainder): 0

Augmented Assignment Operators

Shortcut operators that combine an operation with assignment.

int x = 10; int y = 2; x += y; // Equivalent to: x = x + y x -= y; // Equivalent to: x = x - y x *= y; // Equivalent to: x = x * y x /= y; // Equivalent to: x = x / y x %= y; // Equivalent to: x = x % y

Increment and Decrement Operators

Special operators for increasing or decreasing a value by 1.

int x = 1; x = x + 1; // x becomes 2 x += 1; // x becomes 3 x++; // x becomes 4 (post-increment) ++x; // x becomes 5 (pre-increment) x--; // x becomes 4 (post-decrement) --x; // x becomes 3 (pre-decrement)

Order of Operations (PEMDAS)

Java follows the standard mathematical order of operations: Parentheses, Exponents, Multiplication, Division, Addition, Subtraction.

double result = 3 + 4 * (7 - 5) / 2.0; // Calculation: 3 + 4 * 2 / 2.0 = 3 + 8 / 2.0 = 3 + 4.0 = 7.0

Random Number Generation

Java provides the Random class to generate random numbers, which is useful for games, simulations, and random selections.

import java.util.Random; public class RandomExample { public static void main(String[] args) { Random random = new Random(); int number = random.nextInt(); // Random integer between -2 billion and +2 billion System.out.println(number); number = random.nextInt(1, 6); // Random integer between 1 and 5 (inclusive) System.out.println(number); // To get from 1 to 6 number = random.nextInt(1, 7); System.out.println(number); } }

Random Double Values

Random random = new Random(); double randomDouble = random.nextDouble(); // Between 0.0 and 1.0 System.out.println(randomDouble);

Random Boolean Values

Random random = new Random(); boolean isHeads = random.nextBoolean(); System.out.println(isHeads); if (isHeads) { System.out.println("HEAD"); } else { System.out.println("TAILS"); }

Note: Remember to import java.util.Random at the top of your file when using the Random class.

Simple Math Operations

Java Math Library

Java's Math class provides common mathematical functions and constants that are useful for calculations.

double result; System.out.println(Math.PI); // Value of PI result = Math.pow(2, 5); // 2 to the power of 5: 32 result = Math.abs(-5); // Absolute value: 5 result = Math.sqrt(9); // Square root: 3 result = Math.round(3.14); // Round to nearest integer: 3 result = Math.ceil(3.14); // Round up: 4 result = Math.floor(3.99); // Round down: 3 result = Math.max(10, 20); // Maximum value: 20 result = Math.min(10, 20); // Minimum value: 10

Note: The Math class is part of the java.lang package, which is automatically imported in every Java program, so you don't need to import it explicitly.

Exercise 2: Currency Converter

Question: Create a program that converts between USD and EUR. Ask the user for the amount and the direction of conversion (USD to EUR or EUR to USD), then display the converted amount.

Sample Input:
Enter the amount: 100 Enter conversion direction (1 for USD to EUR, 2 for EUR to USD): 1
Expected Output:
100.0 USD = 92.0 EUR

Note: Use an exchange rate of 1 USD = 0.92 EUR.

import java.util.Scanner; public class CurrencyConverter { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); final double EXCHANGE_RATE = 0.92; // 1 USD = 0.92 EUR System.out.print("Enter the amount: "); double amount = scanner.nextDouble(); System.out.print("Enter conversion direction (1 for USD to EUR, 2 for EUR to USD): "); int direction = scanner.nextInt(); if (direction == 1) { // USD to EUR double convertedAmount = amount * EXCHANGE_RATE; System.out.println(amount + " USD = " + convertedAmount + " EUR"); } else if (direction == 2) { // EUR to USD double convertedAmount = amount / EXCHANGE_RATE; System.out.println(amount + " EUR = " + convertedAmount + " USD"); } else { System.out.println("Invalid conversion direction. Please enter 1 or 2."); } scanner.close(); } }
