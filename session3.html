While Loops

While loops are used when you want to repeat a block of code as long as a condition is true.

Basic While Loop Syntax

while (condition) { // code to be repeated }

Simple While Loop Examples

// Count from 1 to 5 int i = 1; while (i <= 5) { System.out.println("Count: " + i); i++; } // User input until they enter "quit" Scanner scanner = new Scanner(System.in); String input = ""; while (!input.equals("quit")) { System.out.print("Enter something (type 'quit' to exit): "); input = scanner.nextLine(); System.out.println("You entered: " + input); }

Do-While Loops

Do-while loops always execute at least once, then check the condition.

// Input validation with do-while Scanner scanner = new Scanner(System.in); String name; do { System.out.print("Enter your name: "); name = scanner.nextLine(); if (name.isEmpty()) { System.out.println("Name cannot be empty. Try again."); } } while (name.isEmpty()); System.out.println("Hello, " + name);

Important: Always make sure your while loop has a way to end! Otherwise, you'll create an infinite loop.

Exercise: Number Guessing Game

Question: Create a number guessing game where the user tries to guess a secret number between 1-100.

Sample Input/Output:
I'm thinking of a number between 1-100 Enter your guess: 50 Too high! Enter your guess: 25 Too low! Enter your guess: 37 Congratulations! You guessed it in 3 attempts!
import java.util.Scanner; import java.util.Random; public class GuessingGame { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); Random random = new Random(); int secretNumber = random.nextInt(1, 101); // 1-100 int guess; int attempts = 0; System.out.println("I'm thinking of a number between 1-100"); do { System.out.print("Enter your guess: "); guess = scanner.nextInt(); attempts++; if (guess > secretNumber) { System.out.println("Too high!"); } else if (guess < secretNumber) { System.out.println("Too low!"); } else { System.out.println("Congratulations! For Loops

For loops are used when you know exactly how many times you want to repeat a block of code.

Basic For Loop Syntax

for (initialization; condition; update) { // code to be repeated }

Simple For Loop Examples

// Count from 1 to 5 for (int i = 1; i <= 5; i++) { System.out.println("Count: " + i); } // Count down from 5 to 1 for (int i = 5; i >= 1; i--) { System.out.println("Countdown: " + i); } // Even numbers from 2 to 10 for (int i = 2; i <= 10; i += 2) { System.out.println("Even: " + i); }

Break and Continue Statements

// Break - exit loop immediately for (int i = 1; i <= 10; i++) { if (i == 5) { break; // Stop at 5 } System.out.println("Number: " + i); } // Output: 1, 2, 3, 4 // Continue - skip current iteration for (int i = 1; i <= 10; i++) { if (i % 2 == 0) { continue; // Skip even numbers } System.out.println("Odd: " + i); } // Output: 1, 3, 5, 7, 9

Loop Components:

  • Initialization: int i = 1 (start counting at 1)
  • Condition: i <= 5 (continue while i is less than or equal to 5)
  • Update: i++ (increment i by 1 each time)

Arrays

Arrays allow you to store multiple values of the same type in a single variable.

Creating and Using Arrays

// Declaration and initialization int[] numbers = new int[5]; // Array of 5 integers String[] names = {"Alice", "Bob", "Charlie"}; // Array with initial values // Assigning values numbers[0] = 10; // First element numbers[1] = 20; // Second element numbers[2] = 30; // Third element // Accessing values System.out.println(numbers[0]); // Output: 10 System.out.println(names[1]); // Output: Bob // Array length System.out.println("Array length: " + numbers.length);

Looping Through Arrays

int[] scores = {85, 92, 78, 96, 88}; // Using for loop for (int i = 0; i < scores.length; i++) { System.out.println("Score " + (i + 1) + ": " + scores[i]); } // Using enhanced for loop (for-each) for (int score : scores) { System.out.println("Score: " + score); }

Array Methods

import java.util.Arrays; int[] numbers = {5, 2, 8, 1, 9}; // Sort array Arrays.sort(numbers); System.out.println("Sorted: " + Arrays.toString(numbers)); // Fill array with same value int[] filled = new int[5]; Arrays.fill(filled, 42); System.out.println("Filled: " + Arrays.toString(filled));

Methods

Methods allow you to organize your code into reusable blocks.

Void Methods (No Return Value)

public class MethodDemo { // Simple void method public static void printWelcome() { System.out.println("Welcome to Java!"); System.out.println("Enjoy learning!"); } // Method with parameters public static void greetUser(String name, int age) { System.out.println("Hello " + name + "!"); System.out.println("You are " + age + " years old."); } public static void main(String[] args) { printWelcome(); // Call the method greetUser("Alice", 25); // Pass arguments } }

Return Methods

public class ReturnMethods { // Method that returns a value public static double square(double number) { return number * number; } public static String getFullName(String first, String last) { return first + " " + last; } public static boolean isAdult(int age) { return age >= 18; } public static void main(String[] args) { double result = square(5.0); System.out.println("5 squared is: " + result); String fullName = getFullName("John", "Doe"); System.out.println("Full name: " + fullName); boolean adult = isAdult(20); System.out.println("Is adult: " + adult); } }

Method Overloading

public class OverloadingDemo { // Different versions of add method public static int add(int a, int b) { return a + b; } public static int add(int a, int b, int c) { return a + b + c; } public static double add(double a, double b) { return a + b; } public static void main(String[] args) { System.out.println(add(5, 10)); // 15 System.out.println(add(5, 10, 15)); // 30 System.out.println(add(2.5, 3.7)); // 6.2 } }

Variable Scope: Variables declared inside a method are local to that method and cannot be accessed outside.

Varargs Methods (Variable Arguments)

Varargs (variable arguments) allow you to pass a variable number of arguments to a method. This eliminates the need for multiple overloaded methods when you want to handle different numbers of parameters.

Varargs Syntax

// Basic varargs syntax returnType methodName(type... parameterName) { // method body } // Example public static int sum(int... numbers) { int total = 0; for (int num : numbers) { total += num; } return total; }

Basic Varargs Examples

Simple Varargs Method

public class VarargsBasic { // Method that can take any number of int arguments public static int add(int... numbers) { int sum = 0; for (int num : numbers) { sum += num; } return sum; } public static void main(String[] args) { // Calling with different numbers of arguments System.out.println(add(1, 2)); // 3 System.out.println(add(1, 2, 3)); // 6 System.out.println(add(1, 2, 3, 4, 5)); // 15 System.out.println(add()); // 0 (empty) } }

Varargs with Other Parameters

When using varargs with other parameters, the varargs parameter must be the last one:

public class VarargsMixed { // Varargs with other parameters (varargs must be last) public static void printStudents(String course, String... students) { System.out.println("Course: " + course); System.out.println("Students:"); for (String student : students) { System.out.println("- " + student); } System.out.println("Total students: " + students.length); } public static void main(String[] args) { // Print students with different numbers printStudents("Java", "Alice", "Bob", "Charlie"); printStudents("Python", "David"); // Only one student printStudents("Math"); // No students (empty varargs) } }

Varargs vs Arrays

Varargs are treated as arrays inside the method, so you can also pass arrays directly:

public class VarargsArray { public static void printNumbers(int... numbers) { System.out.print("Numbers: "); for (int num : numbers) { System.out.print(num + " "); } System.out.println(); } public static void main(String[] args) { // Using individual arguments printNumbers(1, 2, 3); // Using an array int[] nums = {4, 5, 6, 7}; printNumbers(nums); // Empty call printNumbers(); } }

Complex Varargs Examples

Multiple Data Types with Varargs

public class ComplexVarargs { // Find average of numbers (handles empty case) public static double average(int... numbers) { if (numbers.length == 0) { return 0.0; } int sum = 0; for (int num : numbers) { sum += num; } return (double) sum / numbers.length; } // Find maximum value public static int max(int... numbers) { if (numbers.length == 0) { throw new IllegalArgumentException("No numbers provided"); } int max = numbers[0]; for (int num : numbers) { if (num > max) { max = num; } } return max; } // Format and display messages public static void displayMessages(String header, String... messages) { System.out.println("=== " + header + " ==="); if (messages.length == 0) { System.out.println("No messages to display"); } else { for (int i = 0; i < messages.length; i++) { System.out.println((i + 1) + ". " + messages[i]); } } System.out.println(); } public static void main(String[] args) { // Average examples System.out.println("Average: " + average(10, 20, 30)); // 20.0 System.out.println("Average: " + average(5, 15, 25, 35)); // 20.0 // Max examples System.out.println("Max: " + max(5, 2, 8, 1, 9)); // 9 // Display messages displayMessages("Important Notes", "Learn Java", "Practice daily", "Build projects"); displayMessages("Reminders"); // No messages } }

Important Rules for Varargs:

  • Only one varargs parameter per method
  • Varargs must be the last parameter in the method signature
  • Varargs are treated as arrays inside the method
  • You can pass an array instead of individual values
  • Varargs can be empty (zero arguments)

Varargs vs Method Overloading

Varargs can often replace multiple overloaded methods. Compare these two approaches:

// WITHOUT VARARGS (multiple overloaded methods) public class WithoutVarargs { public static int add(int a, int b) { return a + b; } public static int add(int a, int b, int c) { return a + b + c; } public static int add(int a, int b, int c, int d) { return a + b + c + d; } // Would need more methods for more parameters... } // WITH VARARGS (single method) public class WithVarargs { public static int add(int... numbers) { int sum = 0; for (int num : numbers) { sum += num; } return sum; } // Handles any number of parameters! }
Situation Use Method Overloading Use Varargs
Fixed number of parameters ‚úì Better choice ‚ùå Not necessary
Variable number of parameters ‚ùå Many methods needed ‚úì Perfect fit
Type safety ‚úì Compile-time checking ‚ùå Runtime checking only
Code maintenance ‚ùå Multiple methods to update ‚úì Single method to maintain
Readability ‚úì Clear parameter expectations ‚úì Flexible and concise

Best Practice: Use varargs when you need flexibility in the number of parameters, but prefer method overloading when you have a fixed, small number of well-defined parameter combinations.

Exercise: Flexible Calculator with Varargs

Question: Create a calculator class that uses varargs to handle mathematical operations with any number of inputs.

  • sum(numbers...) - returns sum of all numbers
  • product(numbers...) - returns product of all numbers
  • average(numbers...) - returns average (handle empty case)
  • findMax(numbers...) - returns largest number
  • findMin(numbers...) - returns smallest number
  • joinStrings(separator, strings...) - joins strings with separator
Expected Output:
Sum: 15 Product: 120 Average: 3.0 Max: 5 Min: 1 Joined: Java-Python-C++
public class FlexibleCalculator { public static int sum(int... numbers) { int total = 0; for (int num : numbers) { total += num; } return total; } public static int product(int... numbers) { if (numbers.length == 0) { return 0; } int result = 1; for (int num : numbers) { result *= num; } return result; } public static double average(int... numbers) { if (numbers.length == 0) { return 0.0; } return (double) sum(numbers) / numbers.length; } public static int findMax(int... numbers) { if (numbers.length == 0) { throw new IllegalArgumentException("No numbers provided"); } int max = numbers[0]; for (int num : numbers) { if (num > max) { max = num; } } return max; } public static int findMin(int... numbers) { if (numbers.length == 0) { throw new IllegalArgumentException("No numbers provided"); } int min = numbers[0]; for (int num : numbers) { if (num < min) { min = num; } } return min; } public static String joinStrings(String separator, String... strings) { if (strings.length == 0) { return ""; } StringBuilder result = new StringBuilder(strings[0]); for (int i = 1; i < strings.length; i++) { result.append(separator).append(strings[i]); } return result.toString(); } public static void main(String[] args) { // Test the calculator methods System.out.println("Sum: " + sum(1, 2, 3, 4, 5)); // 15 System.out.println("Product: " + product(1, 2, 3, 4, 5)); // 120 System.out.println("Average: " + average(1, 2, 3, 4, 5)); // 3.0 System.out.println("Max: " + findMax(1, 5, 2, 4, 3)); // 5 System.out.println("Min: " + findMin(1, 5, 2, 4, 3)); // 1 System.out.println("Joined: " + joinStrings("-", "Java", "Python", "C++")); // Java-Python-C++ // Edge cases System.out.println("Empty sum: " + sum()); // 0 System.out.println("Empty average: " + average()); // 0.0 System.out.println("Empty join: '" + joinStrings("-") + "'"); // '' } }
