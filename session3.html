For Loops

For loops are used when you know exactly how many times you want to repeat a block of code.

Basic For Loop Syntax

for (initialization; condition; update) { // code to be repeated }

Simple For Loop Examples

// Count from 1 to 5 for (int i = 1; i <= 5; i++) { System.out.println("Count: " + i); } // Count down from 5 to 1 for (int i = 5; i >= 1; i--) { System.out.println("Countdown: " + i); } // Even numbers from 2 to 10 for (int i = 2; i <= 10; i += 2) { System.out.println("Even: " + i); }

Loop Components:

  • Initialization: int i = 1 (start counting at 1)
  • Condition: i <= 5 (continue while i is less than or equal to 5)
  • Update: i++ (increment i by 1 each time)

Exercise: Multiplication Table

While Loops

While loops are used when you want to repeat a block of code as long as a condition is true.

Basic While Loop Syntax

while (condition) { // code to be repeated }

Simple While Loop Examples

// Count from 1 to 5 int i = 1; while (i <= 5) { System.out.println("Count: " + i); i++; } // User input until they enter "quit" Scanner scanner = new Scanner(System.in); String input = ""; while (!input.equals("quit")) { System.out.print("Enter something (type 'quit' to exit): "); input = scanner.nextLine(); System.out.println("You entered: " + input); }

Do-While Loops

Do-while loops always execute at least once, then check the condition.

int number; do { System.out.print("Enter a positive number: "); number = scanner.nextInt(); } while (number <= 0); System.out.println("Thank you! You entered: " + number);

Important: Always make sure your while loop has a way to end! Otherwise, you'll create an infinite loop.

Exercise: Number Guessing Game

Question: Create a number guessing game where the user tries to guess a secret number between 1-100.

Sample Input/Output:
I'm thinking of a number between 1-100 Enter your guess: 50 Too high! Enter your guess: 25 Too low! Enter your guess: 37 Congratulations! You guessed it in 3 attempts!
import java.util.Scanner; import java.util.Random; public class GuessingGame { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); Random random = new Random(); int secretNumber = random.nextInt(100) + 1; // 1-100 int guess; int attempts = 0; System.out.println("I'm thinking of a number between 1-100"); do { System.out.print("Enter your guess: "); guess = scanner.nextInt(); attempts++; if (guess > secretNumber) { System.out.println("Too high!"); } else if (guess < secretNumber) { System.out.println("Too low!"); } else { System.out.println("Congratulations! You guessed it in " + attempts + " attempts!"); } } while (guess != secretNumber); scanner.close(); } }

Arrays

Arrays allow you to store multiple values of the same type in a single variable.

Creating and Using Arrays

// Declaration and initialization int[] numbers = new int[5]; // Array of 5 integers String[] names = {"Alice", "Bob", "Charlie"}; // Array with initial values // Assigning values numbers[0] = 10; // First element numbers[1] = 20; // Second element numbers[2] = 30; // Third element // Accessing values System.out.println(numbers[0]); // Output: 10 System.out.println(names[1]); // Output: Bob // Array length System.out.println("Array length: " + numbers.length);

Looping Through Arrays

int[] scores = {85, 92, 78, 96, 88}; // Using for loop for (int i = 0; i < scores.length; i++) { System.out.println("Score " + (i + 1) + ": " + scores[i]); } // Using enhanced for loop (for-each) for (int score : scores) { System.out.println("Score: " + score); }

Important: Array indices start at 0! The first element is at index 0, not 1.

String Manipulation

Strings have many useful methods for working with text data.

Common String Methods

String text = "Hello World"; // Length System.out.println("Length: " + text.length()); // 11 // Case conversion System.out.println("Uppercase: " + text.toUpperCase()); // HELLO WORLD System.out.println("Lowercase: " + text.toLowerCase()); // hello world // Substring System.out.println("Substring: " + text.substring(0, 5)); // Hello System.out.println("Substring: " + text.substring(6)); // World // Character at position System.out.println("Char at 0: " + text.charAt(0)); // H // Finding text System.out.println("Contains 'World': " + text.contains("World")); // true System.out.println("Index of 'World': " + text.indexOf("World")); // 6 // Replacement System.out.println("Replace: " + text.replace("World", "Java")); // Hello Java // Trimming whitespace String spaced = " Hello "; System.out.println("Trimmed: '" + spaced.trim() + "'"); // 'Hello'

Looping Through Strings

String word = "Java"; // Loop through characters for (int i = 0; i < word.length(); i++) { char c = word.charAt(i); System.out.println("Character " + i + ": " + c); } // Enhanced for loop with character array for (char c : word.toCharArray()) { System.out.println("Character: " + c); }

Exercise: Palindrome Checker

Question: Create a program that checks if a word is a palindrome (reads the same forwards and backwards).

Sample Input 1:
Enter a word: radar
Expected Output 1:
"radar" is a palindrome!
Sample Input 2:
Enter a word: hello
Expected Output 2:
"hello" is not a palindrome
import java.util.Scanner; public class PalindromeChecker { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Enter a word: "); String word = scanner.nextLine().toLowerCase(); boolean isPalindrome = true; // Check if word is palindrome for (int i = 0; i < word.length() / 2; i++) { if (word.charAt(i) != word.charAt(word.length() - 1 - i)) { isPalindrome = false; break; } } if (isPalindrome) { System.out.println("\"" + word + "\" is a palindrome!"); } else { System.out.println("\"" + word + "\" is not a palindrome"); } scanner.close(); } }

Exercise 3: Password Generator

Question: Create a program that generates a random password of specified length.

Sample Input:
Enter password length: 8
Sample Output:
Your random password: aB3#x9@q
import java.util.Scanner; import java.util.Random; public class PasswordGenerator { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); Random random = new Random(); System.out.print("Enter password length: "); int length = scanner.nextInt(); String characters = "ABCDEFGHIJKLMNOPQRSTUVWXYZ" + "abcdefghijklmnopqrstuvwxyz" + "0123456789" + "!@#$%^&*"; StringBuilder password = new StringBuilder(); for (int i = 0; i < length; i++) { int randomIndex = random.nextInt(characters.length()); password.append(characters.charAt(randomIndex)); } System.out.println("Your random password: " + password.toString()); scanner.close(); } }

Exercise 4: Shopping List Manager

Question: Create a program that allows users to add items to a shopping list and display them.

Sample Input/Output:
=== Shopping List Manager === 1. Add item 2. View list 3. Exit Choose option: 1 Enter item: Milk Item added! Choose option: 1 Enter item: Bread Item added! Choose option: 2 Your Shopping List: 1. Milk 2. Bread Choose option: 3 Goodbye!
import java.util.Scanner; public class ShoppingList { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); String[] shoppingList = new String[10]; // Array to store up to 10 items int itemCount = 0; int choice; do { System.out.println("=== Shopping List Manager ==="); System.out.println("1. Add item"); System.out.println("2. View list"); System.out.println("3. Exit"); System.out.print("Choose option: "); choice = scanner.nextInt(); scanner.nextLine(); // Consume newline switch (choice) { case 1: if (itemCount < shoppingList.length) { System.out.print("Enter item: "); String item = scanner.nextLine(); shoppingList[itemCount] = item; itemCount++; System.out.println("Item added!\n"); } else { System.out.println("Shopping list is full!\n"); } break; case 2: System.out.println("Your Shopping List:"); if (itemCount == 0) { System.out.println("List is empty\n"); } else { for (int i = 0; i < itemCount; i++) { System.out.println((i + 1) + ". " + shoppingList[i]); } System.out.println(); } break; case 3: System.out.println("Goodbye!"); break; default: System.out.println("Invalid choice!\n"); } } while (choice != 3); scanner.close(); } }
